# 地形の近距離ディテール設計

- **日付**: 2026-06-11
- **前提**: 木と草のリアル化（同日マージ済み）の続き
- **目的**: 草の間から見える地面が「ベタ塗りの緑」に見える問題を解消する
- **方針**: terrain.js のフラグメントシェーダ拡張のみで完結させる
  （ジオメトリ・頂点カラー生成は変更しない）

## 問題

地形マテリアルは頂点カラー＋輝度のみのディテールノイズで、
(1) 法線が頂点解像度のまま平坦、(2) 近距離の色むらが明度変化だけで
土・枯れ草の色相変化がない。結果、近くの地面がプラスチック板に見える。

## スコープ

terrain.js の `onBeforeCompile` 拡張のみ:

1. **ディテールバンプ**: 既存 `tnoise` の勾配から微細法線を作り、
   ビュー空間法線に距離フェード付きでブレンドする。
   遠距離はフェードアウトしてノイズのちらつきを防ぐ（〜80m）
2. **土の色むら**: 草色（G が支配的）のピクセルに限り、中周波ノイズの
   パッチで暗い土色（乗算 tint）へ寄せる。砂・岩には適用しない
3. 既存の輝度ディテールは維持

### 非スコープ

- 地形ジオメトリの増分割・ディスプレイスメント
- 水面・空・ポストプロセス

## 検証

Playwright で地表レベルの近接スクショを撮り、草の間の地面に
凹凸と土の色むらが出ていることを確認。遠景のちらつきがないことも確認。
