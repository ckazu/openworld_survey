# スクリーンスペース光芒（サンシャフト）設計

- **日付**: 2026-06-11
- **前提**: リアル化 PR（feature/realism-overhaul）の続き。スタックブランチ
- **目的**: 太陽方向を向いたとき、木々や雲の隙間から光の筋（光芒）が差す
  大気表現を追加し、逆光シーンの空気感を引き上げる
- **方針**: 合成済みフレームに対する放射状ブラーの単一ポストパス
  （occlusion バッファ不要の軽量近似。外部アセット不要）

## スコープ

1. main.js に ShaderPass「LightShaftShader」を追加（Bloom の後・OutputPass の前、
   リニア HDR 空間で動作）
   - 各ピクセルから太陽のスクリーン座標へ向かって 56 サンプルの放射状マーチ
   - 輝度しきい値を超えた明部のみを減衰付きで累積し、加算合成
2. 毎フレーム、太陽のスクリーン座標と強度を更新
   - `camera.position + sunDirection * 1000` を投影して uv を得る
   - 太陽が画面外・背後のときは強度 0 へフェード（アーティファクト防止）

### 非スコープ

- 深度/オクルージョンバッファを使う物理的に正確な god rays
- 太陽ディスク自体の描画変更

## 検証

太陽方向を向いた構図（樹冠越しの逆光）で光芒が見えること、
太陽を背にした構図で何も変化しないこと、画面端での破綻がないことを確認。
