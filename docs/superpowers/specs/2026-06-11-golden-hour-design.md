# ゴールデンアワー＋フォトジェニック化 設計

- **日付**: 2026-06-11
- **前提**: 有機的リアル化（PR #3）マージ済み
- **目的**: 光のムードをゴールデンアワーに変え、写真的な空気感・レンズ感で
  仕上げる（ユーザー選択スコープ: ゴールデンアワー / 朝もや / DOF /
  レンズ効果一式 / 植生の極限密度）

## Step 1: ゴールデンアワー化（sky.js / main.js）

- 太陽高度 26° → 8°。turbidity / rayleigh / mie を上げ地平線を橙〜琥珀に
- 太陽光色を暖色（0xffc587 系）へ、IBL は新しい空から自動再生成
- フォグ色を暖色寄りへ、共有 uniform uSunColor も連動（草・葉の逆光透過が橙に）
- 長い影に合わせてシャドウ範囲・bias、光芒のしきい値・強度を再調整
- Bloom・露出・カラーグレードの再バランス（白飛び・暗部つぶれの防止）

## Step 2: 朝もや（高さフォグ）

- THREE.ShaderChunk の fog チャンク（fog_pars_vertex / fog_vertex /
  fog_pars_fragment / fog_fragment）をグローバルにパッチし、フォグ密度を
  「ワールド高さで指数減衰」させる（湖面・谷に溜まる薄もや）
- 全マテリアル（fog: true のもの）に一括適用される最もプロシージャルな方法
- 副作用（Water・雲ポイント・フォームの fog 挙動）を検証し、破綻時は
  地表ミストビルボード（雲パフ流用）へフォールバック

## Step 3: DOF（main.js）

- BokehPass を控えめ設定（focus 遠め・maxblur 0.004 前後）で追加。
  写真のレンズ感が目的で、ゲーム的視認性は維持する
- 挿入位置はカラーグレードの直前で検証し、破綻すれば Bloom 前に移動

## Step 4: レンズ効果一式（ColorGradeShader 拡張）

- フィルムグレイン（時間アニメ・微量）
- 色収差（画面端のみ RGB をラジアルにずらす・微量）
- シネマトーン（シャドウをわずかに持ち上げ、ハイライトを暖色へ）

## Step 5: 植生の極限密度（任意・計測次第）

- 草クランプ化で見かけ密度は既に約 3 倍のため、Step 1〜4 の完成画で
  不足を感じる場合のみ近距離株数を引き上げる（+30% 上限）
- 相対 FPS 劣化 30% 超で見送り

## 検証

各 Step で定点スクショ（スポーン全景・逆光の森・湖岸）を比較。
ゴールデンアワーは「橙の斜光・長い影・温かい空」が出ているかを確認。
最後に README・ヒーローショットを更新し PR を作成する。
