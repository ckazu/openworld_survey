# ソフトな雲（ビルボードパフ）設計

- **日付**: 2026-06-11
- **前提**: 地形ディテール（同日マージ済み）の続き
- **目的**: 硬い輪郭のジオメトリ雲を、ふんわりした立体感のある雲に置き換える
- **方針**: 放射状フォールオフ × FBM のアルファテクスチャを実行時生成し、
  複数スプライトの重ね合わせで雲を作る（外部画像非依存を維持）

## 問題

現在の雲は IcosahedronGeometry のブロブを Lambert で描いており、
輪郭がポリゴンのまま硬く、画面内で最もローポリ感が強い要素になっている。

## スコープ

1. `textures.js` に `cloudPuffTexture()` を追加
   - 放射状フォールオフに FBM で凹凸をつけたアルファ。RGB は白
2. `ambience.js` の `createClouds` をスプライト方式に置き換え
   - 1 つの雲 = 5〜9 個の `THREE.Sprite`（共有 SpriteMaterial ベース、
     `transparent / depthWrite:false / fog:true`）
   - 下側のパフほど色をわずかに青灰に落として陰を擬似する
   - 既存の流れる更新（x 方向ドリフトとラップ）は維持
3. 水面反射との互換（レイヤー 0 のまま）を維持

### 非スコープ

- レイマーチングによるボリュームクラウド（過剰）
- 空のシェーダ・大気散乱の変更

## 検証

Playwright で空を見上げるスクショと全景スクショを撮り、
輪郭がソフトになったこと・遠近で破綻しないこと・水面反射を確認。
