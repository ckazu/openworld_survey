# 水面の岸辺表現（ショアフォーム）設計

- **日付**: 2026-06-11
- **前提**: ソフト雲（同日マージ済み）の続き
- **目的**: 静的な泡の帯しかない湖岸に「岸へ寄せる波」の動きを加え、
  水面のリアリティを引き上げる
- **方針**: 地形高さをテクスチャに焼いた「ショアマスク」を使う定番手法。
  外部アセット非依存・water.js 内で完結

## スコープ

1. **ショアマスク**: 湖周辺（500m 四方）の `terrainHeight - WATER_LEVEL` を
   256×256 の DataTexture に焼く（初期化時に一度）
2. **フォームプレーン**: 水面+2cm の透明プレーン。フラグメントシェーダで
   - 水際の白い縁（静的、既存の頂点カラー泡帯より細くシャープ）
   - 浅瀬を岸へ進む波頭（sin(深さ×k − time) のバンド、ノイズで切れ目）
   を描く。深い場所・陸地ではアルファ 0
3. **Water の質感調整**: distortionScale / size / waterColor を微調整し
   反射の硬さを和らげる

### 非スコープ

- 屈折・深度ベースの水色グラデーション（コンポーザの depth texture が必要）
- 川・海など湖以外の水域

## 検証

Playwright で湖岸のスクショ（静止比較は 2 時刻）を撮り、
波頭が岸へ進んで見えること・陸へのはみ出しがないこと・遠景の破綻なしを確認。
